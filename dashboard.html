📋 Analysis Overview

This comprehensive dashboard presents Japan's visa issuance patterns from 2006 to 2017. The analysis provides insights into how visa distribution has evolved over time across different continents and countries.

🔬 Data Processing

Raw data has been cleaned and standardized using fuzzy matching for country names and consolidated by continent classifications.

📊 Visualizations

Three interactive Plotly visualizations provide different perspectives on visa trends and distributions.

🎯 Key Insights

Explore how visa patterns vary by continental regions and identify leading countries in visa applications.

💾 Data Source

File: visa_number_in_japan.csv

Records: Multiple entries per year, country

🛠️ Technologies

Processing: Apache Spark

Visualization: Plotly

📁 Output Files

3 interactive HTML visualizations plus cleaned CSV data
